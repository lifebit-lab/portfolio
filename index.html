家計簿PWA

夫婦2人で管理できる家計管理アプリ

OCRでレシートを自動読み込みし、カテゴリ別・月別グラフで 支出を可視化。招待コードでパートナーと家計をリアルタイム共有できます。

Next.js TypeScript Supabase PWA Recharts
使い方
  1. アカウントを作成してログイン
  2. 「家計を新規作成」または招待コードでパートナーと参加
  3. レシートをカメラで撮影 → OCRで自動入力
  4. ダッシュボードで月次グラフ・収支サマリーを確認

医療経営士試験対策PWA

医療経営士3級合格を目指す問題演習アプリ

7つの公式カテゴリを網羅した問題演習・模擬試験機能。 分野別正答率グラフで苦手分野を把握し、効率よく学習できます。

Next.js TypeScript Supabase PWA
使い方
  1. アカウントを作成してログイン
  2. 「練習問題」「模擬試験」「学習履歴」からモードを選択
  3. 分野別・ランダム・頻出問題から学習スタイルを選ぶ
  4. 問題を解いて解説を確認、成績グラフで進捗を追跡

株式AIコンパス

日本株を財務・テクニカルでAIスコアリング

財務健全性・成長性・テクニカル指標を統合した100点満点スコアで 日本株を評価。下落が市場全体の問題か個別企業の問題かをAIが自動判定します。

Next.js TypeScript Supabase J-Quants API Recharts
使い方
  1. 銘柄コード（例：7203）を入力して検索
  2. 財務健全性・成長性・収益性・テクニカルの各スコアを確認
  3. 下落理由（市場調整 vs 企業固有問題）の自動判定を参照
  4. スクリーニングでROE・PER・配当利回りから銘柄を絞り込む

※投資助言ではなく判断材料の提供を目的としています
