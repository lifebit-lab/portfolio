01

家計簿PWA

夫婦2人で管理できる家計管理アプリ

OCRでレシートを自動読み込みし、カテゴリ別・月別グラフで 支出を可視化。招待コードでパートナーと家計をリアルタイム共有できます。

Next.js TypeScript Supabase PWA Recharts
使い方
  1. アカウントを作成してログイン
  2. 「家計を新規作成」または招待コードでパートナーと参加
  3. レシートをカメラで撮影 → OCRで自動入力
  4. ダッシュボードで月次グラフ・収支サマリーを確認

医療経営士試験対策PWA

03

株式AIコンパス

日本株を財務・テクニカルでAIスコアリング

財務健全性・成長性・テクニカル指標を統合した100点満点スコアで 日本株を評価。下落が市場全体か個別企業の問題かをAIが自動判定します。

Next.js TypeScript Supabase J-Quants API Recharts
使い方
  1. 銘柄コード（例：7203）を入力して検索
  2. 財務健全性・成長性・収益性・テクニカルの各スコアを確認
  3. 下落理由（市場調整 vs 企業固有問題）の自動判定を参照
  4. スクリーニングでROE・PER・配当利回りから銘柄を絞り込む

※投資助言ではなく判断材料の提供を目的としています

Okuriba

05

Aurora Memoria

記憶の極光 — インタラクティブアート

夜空に触れると記憶が光になる。マウス・タッチで描いた線がオーロラのように漂い、 時間とともにゆっくり薄れていく没入型インタラクティブ体験。 Web Audio API によるアンビエント音響も内蔵。

Three.js WebGL GLSL Shader Web Audio API
使い方
  1. 夜空をクリック・なぞってオーロラを描く
  2. 下の欄にことばを入力すると光のログに浮かぶ
  3. 記憶は約150秒でゆっくり薄れていく
  4. 「心地の風」ボタンで全記憶を解放
